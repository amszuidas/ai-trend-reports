## 今日最重要的趋势

1. **Agent 竞争从模型能力上移到系统控制层。** Harness、loops、skills、长期记忆、权限审计和回滚能力正在成为产品差异化核心。
2. **AI 开发工具进入规模化分发阶段。** Codex/ChatGPT Work 的用户增长是强信号，但与 Claude Code 的比较仍受活跃周期和产品口径差异影响。
3. **模型公司继续重资产化。** DeepSeek 融资传闻指向数据中心、芯片采购和 Agent 研发，算法效率之外，资本开支能力正成为新门槛。
4. **语音正在从聊天入口升级为执行入口。** 实时推理、工具调用、自然打断和全双工交互开始被集成到同一个语音 Agent。
5. **AI Coding 的价格压力继续增大。** 免费模型加桌面 Agent 的组合会压低获客门槛，但厂商内部基准和单次媒体实测尚不足以证明领先地位。
6. **具身智能评价转向真实工站。** 成功率、连续运行、柔性物料处理、生产系统接入和人工接管率比实验室动作展示更重要。

## 方法与口径

- 窗口以上次成功运行记忆中的连续覆盖点为起点，不以自然日或 13:00 强行截断。
- 每个来源由独立 Subagent 调研，并实际打开正文、公开预览或可核验的公开镜像；没有只依赖搜索摘要。
- 入选依据综合官网位置、发布时间、公开互动、引用传播、事件重要性和信息密度；没有互动数字时不作杜撰。
- 付费文章只总结公开预览，未绕过付费墙。
- 厂商供稿、内部基准、融资传闻、成功率及用户数均按其证据性质明确标注。
- 全程未使用机器之心。
