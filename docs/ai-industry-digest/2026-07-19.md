### 【量子位】让VLA更懂接触，优理奇UniTac让机器人拥有“触觉想象力”

- 作者：henry
- 发布时间：2026-07-18 18:55:47（Asia/Shanghai）
- [原文链接](https://www.qbitai.com/2026/07/453897.html)

- 入选依据：量子位 RSS 与首页前列文章；[arXiv:2606.31451](https://arxiv.org/abs/2606.31451)明确标注被 ECCV 2026 接收，形成可核验学术信号。文章页未显示互动数。

#### 摘要

文章介绍 UniX AI、浙江大学及 MIT、牛津、耶鲁、上海交大等团队提出的 UniTac：一个统一进行跨传感器触觉理解与生成的多模态模型。它把触觉拆为物体属性与传感器配置两层，通过属性描述、传感器识别、触觉重建和对齐训练，学习不同视觉式触觉传感器之间可复用的表示；随后根据视觉信息预测接触后的触觉状态，为 VLA 在真正接触前提供先验。报道给出理解与生成指标，并展示柔性杯、毛巾等操作案例，核心主张是把触觉从事后反馈前移为第一次动作前的风险预判，而非取代接触后的实时反馈。

#### 关键观点与事实

- **【可核验事实】** arXiv 页面列出 11 位作者，并明确标注论文已被 ECCV 2026 接收。
- **【文章陈述】** UniTac 同时学习物体硬度、粗糙度等属性和传感器配置，以缓解同一物体在不同触觉设备上信号差异过大的问题。
- **【文章陈述】** UniTac-7B 在 PHYSICLEAR-Test 得 66.51；四类传感器生成任务平均 SSIM、PSNR 为 0.836、19.93，均为论文／文章报告值，未由本次调研复现实验。
- **【文章陈述】** 预测的触觉状态被接入 VLA，用于触碰前规划抓取位置和力度；事后实时触觉反馈仍然保留。

**行业意义：** **【主 Agent 推断】** VLA 的瓶颈正从“看见并规划轨迹”深入到“如何安全接触”。若统一触觉表示能跨硬件稳定泛化，它可能缓解具身数据碎片化并服务精细装配、易损物抓取和护理机器人；下一步关键是跨本体、跨传感器和长时任务的独立复现。

量子位在 WAIC 现场体验了商汤日日新 SenseNova U1 Pro。报道将它定位为面向复杂多模态任务的“成品交付系统”，不只一次生成图片，还试图连续完成理解目标、规划版式、组织信息、生成、检查与修正。作者用 WAIC 九周年长卷、二十四节气长图、实验室海报、古风场景和电影海报检验 8K 长画幅、文字清晰度与复杂构图。文章还解释其以 32×32 大 Patch 降低视觉 Token 数，再用重叠 Patch、自适应 Noise Control 等办法补细节，同时坦承等待时间、成本、稳定性、图层和团队协作仍待真实项目验证。

#### 关键观点与事实

- **【文章陈述】** U1 Pro 的主要卖点是原生 8K 输出、图文交错式连续创作，以及从“生成”走向“成品交付”。
- **【文章陈述】** 相较常见的 16×16 Patch，32×32 Patch 可把视觉 Token 数降至约四分之一；团队再以 Patch 重叠、噪声控制、采样和损失设计补偿细节。
- **【文章陈述】** 模型在量子位现场测试中能处理超长画幅、较多文字和复杂风格，但这些测试不是独立标准化评测。
- **【可核验事实】** 商汤公开产品材料列出 U1 Pro，并写明最高支持 8K 分辨率及特殊长宽比。

**行业意义：** **【主 Agent 推断】** 图像模型竞争正从单张观感转向复杂任务完成度和可交付性。如果闭环修正与高分辨率排版能在真实设计流程稳定复现，视觉生成会更接近内容生产 Agent；当前证据主要来自厂商发布与媒体现场体验，尚不足以证明跨场景、成本和长期一致性领先。

## 方法与口径

- 四个来源由四个独立 Subagent 分批调研，每个 Subagent 仅负责一个来源；最多同时运行三个来源 Subagent。
