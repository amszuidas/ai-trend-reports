### 【量子位】高德发布通用世界模型工坊 ABot-World Studio：5090 单卡可生成小时级实时交互式视频与 3D 场景

- 作者：量子位的朋友们；稿件由高德地图提供
- 发布时间：2026-07-14 11:46:35（Asia/Shanghai）
- 来源：量子位
- [原文链接](https://www.qbitai.com/2026/07/449568.html)
- [官方代码](https://github.com/amap-cvlab/ABot-World)
- 入选依据：抓取时位于首页时间流前列；兼具可测试产品和开放代码。抓取时官方 GitHub 显示 437 Stars、13 Forks，属于可见互动信号。

#### 摘要

ABot-World Studio 将动作条件交互视频和 3DGS 场景生成整合进同一工作台，用户可以通过文本或图片生成可实时操控、保存和分享的 AI 世界。官方称系统可在单张 RTX 5090 上本地运行，连续推理超过一小时，并用“任意门”连接多个 3D 世界。底层包含 ABot-World0 与 ABot-3DWorld0，并采用长时序误差控制及“生成—评估—修复”机制。模型、推理代码和本地演示已开放。

#### 关键观点与事实

- **【可核验事实】** 官方仓库采用 Apache-2.0 许可证，开放模型、推理代码与本地 Gradio Demo。
- **【可核验事实】** 仓库列出的测试配置为 RTX 5090、720p、16 FPS、1.2 秒延迟和约 19GB 显存。
- **【文章陈述】** 连续推理超过一小时且没有质量衰减，来自官方自测。
- **【文章陈述】** Studio 可同时输出交互视频与具几何结构的 3DGS 资产。
- **【主 Agent 推断】** 消费级单卡和开放代码降低了原型验证门槛，但长期一致性仍需独立复现。

**对行业的意义：** 世界模型正从固定时长生成走向可部署的实时模拟器，可服务机器人仿真、游戏、影视预演和空间内容生产。
