# AI Trend Reports

每日整理值得关注的 AI / Agent 开源项目与 AI 行业进展。

## 内容栏目

- **AI / Agent 项目日报**：追踪 GitHub Trending 中的 AI / Agent 项目，并核验其功能、成熟度与边界。
- **AI 行业文章日报**：汇总重要行业文章、融资、模型、产品与基础设施进展。
- **中国早期 AI 初创求职雷达**：跟踪适合 Agent Harness / Runtime 工程师加入的大陆早期 AI 应用公司，并核验融资、团队规模、招聘与产品进展。

使用顶部导航进入栏目，或使用搜索框检索项目、公司和关键词。
